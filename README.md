# Seoul Insider — Image Hosting

This repository hosts images for the **Seoul Insider** multi-country blog network published on Blogger.

## Structure

```
images/
└── YYMMDD/         # publication date
    ├── 01.png      # cover image
    ├── 02.png
    └── ...07.png
```

## Usage

Images are referenced from blog posts via the GitHub raw URL pattern:

```
https://raw.githubusercontent.com/deok22-cmd/AdSense/main/images/{YYMMDD}/{NN}.png
```
